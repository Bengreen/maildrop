Mail - Enabling the Script Menu

Before you can use Maildrop from Mail you need to enable the system wide scripts menu:

  1. Open a new Finder window.
  2. Navigate to the Applications folder
  3. Open the Apple Scripts folder
  4. Run the AppleScript Utility program
  5. Make sure the Show Script Menu in menu bar option is checked
