Mail - Add an Email Activity

You can easily take an email and create a completed Email Activity in Salesforce.com.

Reminder: If you can't see the scripts menu, remember you need to enable the system wide scripts menu.



  1. Select the relevant email in Mail.
  2. From the scripts menu, select "Add Email to Salesforce.com"
  3. Login when prompted. (You'll only need to login the first time after Maildrop starts)
  4. The Add Email window will open to allow you to specify who and what the Email is related to.

    See Add Email Window for more details
  5. Click create, and the created activity will be opened in the browser.
