Entourage - Create a Case from an Email

You can easily create a new case in Salesforce.com from the selected email in Entourage.




  1. Select the relevant email in Entourage.
  2. From the Button bar, select "Add Case"
  3. Login when prompted. (You'll only need to login the first time after Maildrop starts)
  4. The Case is created and opened in your browser. If there is a contact with a matching email address, then the case will be automatically associated with that contact.
