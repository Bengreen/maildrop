Maildrop and AppleScript

Previous versions of Maildrop required you to use the AppleScripts menu, either the one built into Entourage, or the system wide scripts menu for Mail.

Starting with Maildrop 2.0, the button bar is the primary way to interact with Maildrop, however it still uses AppleScript to integrate to Entourage and Mail under the covers, and if you wish, you can continue to use the scripts menu to interact with Maildrop. Revised versions of the AppleScripts used are copied to the relevant directories as part of the first run of Maildrop 2.0.

In addition other System wide, or application specific enhancements to run AppleScript can continue to be used, for example, QuickSilver or Mail Act-On.

Enabling the system wide scripts menu:

  1. Open a new Finder window.
  2. Navigate to the Applications folder
  3. Open the Apple Scripts folder
  4. Run the AppleScript Utility program
  5. Make sure the Show Script Menu in menu bar option is checked
